==================================
Welcome to CHARMM's documentation!
==================================

You are now browsing CHARMM 37b1 documentation. This is a personnally maintained documentation based on the `official documentation <http://www.charmm.org/documentation/c37b1/index.html>`_ by `Sunhwan Jo <http://sunhwanj.com>`_. The original CHARMM documentations are translated into `reStructuredText <http://docutils.sourceforge.net/rst.html>`_ and compiled to HTML by `Sphinx <http://sphinx-doc.org>`_. During the translation, many format has to be adjusted by hand, so please let me know if you found any error or misformatted page.
